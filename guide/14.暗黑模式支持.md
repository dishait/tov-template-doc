# 暗黑模式支持

暗黑模式由 [vue-dark-switch](https://github.com/dishait/vue-dark-switch) 实现。

```html
<script setup>
import { SwitchIcon } from "vue-dark-swicth"
</script>

<template>
	<!-- 暗黑 switch，一键切换暗黑模式 -->
	<SwitchIcon /> 
</template>
```

具体可见 👉 [vue-dark-switch](https://github.com/dishait/vue-dark-switch)
